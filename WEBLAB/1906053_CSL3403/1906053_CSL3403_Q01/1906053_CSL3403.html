Difference between HTML tags and HTML Attributes

HTML Tag HTML Attribute
  1. Tags are also used to markup the start and end of an HTML element .
  1. An attribute is used to define the characteristics of an HTML element .
  1. Tags are the building blocks of a web page.
  1. Attribute enhances the functionality of a tag.
  1. Tags do not appear in the browser window but they affect the display of the text and non-text items in it.
  1. An attribute is always specified inside the opening tag. All attributes consist of two parts- a name and a value.
